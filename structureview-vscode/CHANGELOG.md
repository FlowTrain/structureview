# Changelog

## [0.1.0] — 2026

### Added
- Markdown preview with full GFM support — headings, tables, code blocks, task lists, blockquotes
- JSON interactive tree viewer — collapsible nodes, syntax coloring, auto-collapse for large objects
- Live preview — updates as you type without saving
- Scroll sync between editor and preview
- In-preview search with Ctrl+F, prev/next navigation, match count
- Raw source toggle (Ctrl+`)
- Automatic VS Code theme support — light, dark, high-contrast
- Works in Cursor, Windsurf, and any VS Code-compatible editor
- Right-click context menu in editor and Explorer
- Editor title bar preview button
- Configurable auto-preview, scroll sync, default view mode
